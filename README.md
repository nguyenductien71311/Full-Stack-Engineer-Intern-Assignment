# Photo-upload-and-commnet

## Features
- Upload photos
- View and comment on photos

## Tech Stack
- **Frontend:** Next.js, TypeScript, Ant Design (antd)
- **Backend:**  Next.js API routes
- **Database:** PostgreSQL with Prisma ORM

## Deployment
